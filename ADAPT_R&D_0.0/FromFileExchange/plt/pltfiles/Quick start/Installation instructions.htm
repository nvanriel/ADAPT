Installation instructions

 

 
  • Create a new folder called plt in any convenient place on your disk.
  • Add this new folder to your Matlab path.
  • Extract the downloaded archive file (plt.zip) into this new folder. It is important that your unzipper preserves the directory structure. (Most unzippers will do this properly by default.) After the unzip operation the new folder will contain the files and folders shown below. The 22 example programs will appear in a subdirectory called demo. It may be convenient to also add this demo directory to your Matlab path. If you prefer not to do that, you can still run the demo programs by using the Matlab cd command to make the demo folder active.

Notes for PC based systems:

If you type plt help or if you click on the "Help" tag in the plt menu box, then plt.chm is opened with the Windows help system hh.exe. (Normally hh.exe resides in the C:\Windows folder) If hh.exe is missing or if you prefer not to use the chm compiled help file, just delete the chm file and plt will instead open plt.htm using your browser.

Notes for Unix based systems:

If you type plt help or if you click on the "Help" tag in the plt menu box, then plt.htm is opened in your browser. This file points to many other html files and images inside the pltfiles\ folder. If your browser supports chm help files and you prefer to use it, you can do that by deleting or renaming the plt.htm file.
